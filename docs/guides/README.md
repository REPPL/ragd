# Guides

Task-oriented how-to documentation.

## Purpose

Guides help users accomplish specific tasks. Each guide focuses on a practical goal and provides step-by-step instructions.

## Guide Guidelines

When writing guides:
- Focus on accomplishing a task
- Start with the goal, not the background
- Provide clear, numbered steps
- Include troubleshooting tips
