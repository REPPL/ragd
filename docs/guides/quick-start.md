# Quick Start Card

**The 4 commands you need to know.**

---

## 1. Set Up

```bash
ragd init
```

Detects your hardware, recommends models, creates configuration.

---

## 2. Add Documents

```bash
ragd index ~/Documents/
```

Indexes PDFs, Markdown, text, and HTML files.

---

## 3. Search

```bash
ragd search "your question here"
```

Finds relevant excerpts from your documents.

---

## 4. Chat

```bash
ragd chat
```

Interactive Q&A with your knowledge base (requires Ollama).

---

## Quick Reference

| Task | Command |
|------|---------|
| Check status | `ragd info` |
| Health check | `ragd doctor` |
| List documents | `ragd list` |
| Get help | `ragd --help` |
