# CLI Reference

For complete command specifications, see the **[CLI Reference](../../reference/cli-reference.md)** in the Reference documentation.

## Quick Reference

The most commonly used commands:

| Command | Description |
|---------|-------------|
| `ragd init` | Initialise ragd configuration |
| `ragd index <path>` | Index documents |
| `ragd search <query>` | Search knowledge base |
| `ragd ask <question>` | Ask a question (LLM-powered) |
| `ragd chat` | Interactive chat mode |
| `ragd info` | Show index status |
| `ragd doctor` | Run health checks |

## Global Options

| Option | Description |
|--------|-------------|
| `--help`, `-h` | Show help |
| `--version`, `-V` | Show version |
| `--format`, `-f` | Output format: `rich`, `plain`, `json` |
| `--verbose`, `-v` | Verbose output |
| `--quiet`, `-q` | Minimal output |
