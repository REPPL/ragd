# Feature Roadmap

Feature-centric planning for ragd development.

## Structure

| Directory | Contains | When to Move |
|-----------|----------|--------------|
| [active/](./active/) | Currently implementing | Move to completed/ when done |
| [planned/](./planned/) | Next up for development | Move to active/ when started |
| [completed/](./completed/) | Finished features | Archive from active/ |

## Workflow

1. **Plan:** Create feature document in `planned/`
2. **Start:** Move to `active/` when development begins
3. **Complete:** Move to `completed/` when feature is released

## Feature Document Template

```markdown
# Feature: [Name]

## Summary
Brief description of the feature.

## Motivation
Why this feature is needed.

## Requirements
- [ ] Requirement 1
- [ ] Requirement 2

## Design
High-level design approach.

## Status
[Planned/In Progress/Completed]
```
