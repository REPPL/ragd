# F-094: Use Case Gallery

**Status:** Completed
**Milestone:** v0.8.7

## Problem Statement

Users don't know what ragd can do for them. Real-world examples inspire and guide adoption.

## Design Approach

Create a gallery of use cases with complete examples showing ragd solving real problems.

### Gallery Structure
```
docs/use-cases/
├── README.md                    # Use case index
├── personal-notes.md            # Managing Obsidian/notes
├── research-papers.md           # Academic research
├── code-documentation.md        # Code and docs
├── legal-documents.md           # Contracts, policies
├── recipe-collection.md         # Personal recipes
└── meeting-notes.md             # Work meeting notes
```

### Use Case Format
1. **Scenario** - Who you are, what you need
2. **Setup** - How to configure ragd
3. **Workflow** - Step-by-step usage
4. **Example queries** - Real query examples
5. **Tips** - Power user suggestions
