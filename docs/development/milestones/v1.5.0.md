## Focus Areas

### REST API (F-133)
```bash
# Start API server
ragd serve --port 8080

# Endpoints
POST /api/v1/index          # Index documents
GET  /api/v1/search         # Search
POST /api/v1/chat           # Chat
GET  /api/v1/status         # Status
GET  /api/v1/documents      # List documents
```

### OpenAPI Documentation
- Auto-generated from endpoints
- Interactive Swagger UI
- Client SDK generation

### Onboarding Flow (F-134)
```bash
ragd init
# Welcome to ragd!
#
# Detected: Apple M2, 16GB RAM
# Recommended model: llama3.2:3b
#
# ? Install recommended model? [Y/n]
# ? Index a sample document? [Y/n]
# ? Run a test search? [Y/n]
#
# Setup complete! Try: ragd search "hello"
```

### Health Endpoints (F-137)
```bash
# Kubernetes-ready health checks
GET /health      # Always returns 200 if running
GET /ready       # Returns 200 when ready to serve
GET /metrics     # Prometheus-compatible metrics
```
