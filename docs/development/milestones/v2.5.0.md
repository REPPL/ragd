## Focus Areas

### Plugin System (F-200)

Extend ragd with custom plugins:

```bash
# Install community plugin
ragd plugin install ragd-plugin-notion

# List installed plugins
ragd plugin list

# Create custom plugin
ragd plugin init my-plugin --template processor
```

**Plugin Types:**
- **Processors** - Custom document processors
- **Retrievers** - Alternative retrieval strategies
- **Exporters** - Custom export formats
- **Integrations** - External service connectors

### Advanced Dashboard (F-202)

Web-based analytics and visualisations:

- Document statistics and trends
- Search query analytics
- Storage usage monitoring
- Performance metrics

### Knowledge Graph Explorer (F-203)

Visual navigation of your knowledge:

```bash
# Launch graph explorer
ragd graph explore --ui

# Open http://localhost:8080/graph
```

- Interactive node navigation
- Relationship discovery
- Entity clustering
- Export to common graph formats

### Saved Searches (F-204)

Bookmark and reuse searches:

```bash
# Save a search
ragd search "machine learning papers" --save "ml-papers"

# Run saved search
ragd search --load "ml-papers"

# List saved searches
ragd search --list-saved
```

### Export Templates (F-205)

Custom export formats:

```bash
# Create template
ragd export template create "research-report" --format markdown

# Use template
ragd export --template "research-report" --query "recent findings"
```
