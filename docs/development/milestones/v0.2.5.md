## Draft Release Notes

### ragd v0.2.5 - HTML Enhancement

**Beyond Archives.**

Building on v0.2's SingleFile support, v0.2.5 brings advanced HTML processing to all your web content.

**New Features:**

- **Fast Parsing** - selectolax integration for 10-100x faster HTML parsing
- **Rich Metadata** - Automatic extraction of Open Graph, JSON-LD, Schema.org
- **Structure Preservation** - Tables become searchable Markdown, headings preserved
- **Smart Chunking** - Structure-aware chunking respects document boundaries

**Installation:**

```bash
pip install 'ragd[web]'  # For selectolax + trafilatura
```

**Usage:**

```python
from ragd.ingestion.extractor import AdvancedHTMLExtractor

extractor = AdvancedHTMLExtractor()
result = extractor.extract(path_to_html)

# Rich metadata available
print(result.metadata['og_type'])
print(result.metadata['schema_type'])
```

**What's Next:**

v0.3 brings advanced search: hybrid retrieval, reranking, and smarter query processing.
