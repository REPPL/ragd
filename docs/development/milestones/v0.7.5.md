# Milestone: v0.7.5 - Privacy Polish + Tagging (Superseded)

This milestone was superseded by v0.8.x which implemented all planned features.

## Original Goal

Extend the v0.7.0 privacy foundation with data sensitivity tiers, and implement a comprehensive tagging system with provenance tracking, auto-suggestions, and smart collections.

## Theme

**"Organise and Protect"** - Intelligent organisation meets privacy controls.

## Features (Implemented in v0.8.x)

All features originally planned for v0.7.5 were implemented in the v0.8.x series:

|---------|-------------|--------|----------------|
| [F-018](../features/completed/F-018-data-tiers.md) | Data Sensitivity Tiers | ✅ Complete | v0.8.0 |

### Tagging System → v0.8.0/v0.8.1

## Why Superseded

The v0.7.5 milestone was skipped to consolidate the "Intelligence & Organisation" theme into v0.8.x. This allowed:

1. Better feature grouping (tagging + intelligence together)
2. Incremental releases (v0.8.0 → v0.8.1 → v0.8.2 → v0.8.5)
3. Faster delivery of related features
