## Draft Release Notes

### ragd v0.7.5 - Privacy Polish + Tagging

**Organise your knowledge, protect your data.**

Building on v0.7.0's privacy foundation, this release adds data sensitivity tiers and a comprehensive tagging system.

**New Features:**

- **Data Sensitivity Tiers** (F-018) - Classify documents by sensitivity level
- **Tag Provenance** (F-064) - Track how and when tags were applied
- **Auto-Tag Suggestions** (F-061) - Intelligent tag recommendations
- **Tag Library Management** (F-062) - Organise tags with hierarchies and aliases
- **Smart Collections** (F-063) - Dynamic document collections based on rules

**Getting Started:**

```bash
# Set document sensitivity tier
ragd tier set document.pdf confidential

# View tag history
ragd tag history document.pdf

# Get tag suggestions
ragd tag suggest document.pdf

# Create smart collection
ragd collection create "Work Projects" --rule "tag:work AND modified:>30d"
```
