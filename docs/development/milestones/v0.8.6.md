## Focus Areas

### Input Sanitisation
- Validate all CLI inputs
- Prevent path traversal in file operations
- Memory-safe operations for large documents

### Secrets Management (F-083)
- API keys via environment variables (`RAGD_OLLAMA_API_KEY`)
- Encrypted secrets in config (`ragd config set-secret`)
- Never log secrets, even in debug mode
- Secrets masked in `ragd config show`

### Security Audit
- Dependency scan with pip-audit
- Static analysis with bandit
- SBOM generation for supply chain security

## Success Criteria

- [ ] No path traversal vulnerabilities
- [ ] All secrets properly protected
- [ ] 85% test coverage achieved
- [ ] Clean dependency audit
- [ ] SBOM generated and documented
