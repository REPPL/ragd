## Focus Areas

### Technology Stack
- **Backend**: FastAPI (builds on v1.5 API)
- **Frontend**: HTMX + Tailwind (minimal JavaScript)
- **Deployment**: Single process

### WebUI Features

**Search Interface (F-138)**
- Query input with filters
- Results with highlighting
- Faceted navigation by tags/types
- Citation display

**Chat Interface (F-139)**
- Conversation history
- Context display
- Source citations
- Model selection

**Index Interface (F-140)**
- Drag-and-drop upload
- Progress indicators
- Batch upload support
- Status monitoring

### Responsive Design (F-141)
- Mobile-first approach
- Touch-friendly controls
- Adaptive layouts

## Success Criteria

- [ ] WebUI accessible at localhost:8080
- [ ] Search, chat, index functional
- [ ] Mobile-responsive design
- [ ] Single-process deployment
- [ ] Non-technical user tested
