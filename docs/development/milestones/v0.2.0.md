# Milestone: v0.2.0 - Killer Feature

Processing messy real-world documents that most RAG systems struggle with.

## Goal

Extend ragd to handle "messy" documents: scanned PDFs, complex layouts, tables, and multi-column text. This is the differentiating feature that moves ragd from a basic RAG tool to a practical knowledge management system.

## Theme

**"Handle the Real World"** - Documents aren't always clean.

## Draft Release Notes

### ragd v0.2.0 - Killer Feature

**Handle the real world.**

Your documents aren't all clean PDFs. Now ragd can handle the messy ones too.

**New Features:**

- **Scanned PDF Support** - OCR for scanned documents
- **Table Extraction** - Tables become searchable content
- **Complex Layouts** - Multi-column, headers, footers handled
- **Metadata Management** - Add, edit, search document metadata
- **Export** - Backup your knowledge base
- **Watch Folder** - Automatic indexing when documents change
- **Web Archive Support** - Index saved web pages with reader view

**Getting Started:**

```bash
ragd index ~/Documents/scanned-contracts/
ragd search "what are the payment terms"
ragd export --format json backup.json
```

**What's Next:**

v0.3 brings advanced search: hybrid retrieval, reranking, and smarter query processing.
