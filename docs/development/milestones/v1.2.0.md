## Focus Areas

### Onboarding Flow (F-134)

Interactive setup wizard for new users:

```bash
ragd init
# Welcome to ragd!
#
# Detected: Apple M2, 16GB RAM
# Recommended model: llama3.2:3b
#
# ? Install recommended model? [Y/n]
# ? Index a sample document? [Y/n]
# ? Run a test search? [Y/n]
#
# Setup complete! Try: ragd search "hello"
```

**Key Features:**
- Hardware detection and model recommendations
- Optional sample document indexing
- Interactive test to verify installation
- Clear next steps for users

### Target Users

- First-time ragd users
- Non-technical users
- Users migrating from other RAG tools

## Success Criteria

- [ ] `ragd init` guides users through setup
- [ ] Hardware detection works on macOS, Linux, Windows
- [ ] Model recommendations are appropriate for detected hardware
- [ ] Sample document indexing is optional
- [ ] Clear error messages if requirements not met
