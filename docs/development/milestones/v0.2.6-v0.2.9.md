# Milestone: v0.2.6-v0.2.9 - CLI Enhancements

Exposing v0.2.0 backend features through the command-line interface.

## Goal

Complete the v0.2 series by adding CLI commands for all backend features implemented in v0.2.0. This transforms ragd from a library with CLI into a fully CLI-driven application.

## Theme

**"Complete the Interface"** - Every backend feature gets a CLI command.

## New Commands

### v0.2.6: Reindex Command

```bash
ragd reindex --all              # Re-index all documents
ragd reindex --type pdf         # Re-index only PDFs
ragd reindex doc-123            # Re-index specific document
```

### v0.2.7: Metadata Commands

```bash
# Metadata management
ragd meta show doc-123
ragd meta edit doc-123 --title "New Title"

# Tag management
ragd tag add doc-123 important
ragd tag remove doc-123 draft
ragd tag list --counts

# Document listing
ragd list --tag important --project Research
```

### v0.2.8: Export/Import Commands

```bash
ragd export ~/backup.tar.gz
ragd export ~/ml.tar.gz --tag "topic:ml"
ragd import ~/backup.tar.gz --dry-run
ragd import ~/backup.tar.gz --overwrite
```

### v0.2.9: Watch Commands

```bash
ragd watch start ~/Documents
ragd watch start ~/PDFs --pattern "*.pdf"
ragd watch stop
ragd watch status
```

## CLI Architecture

```
ragd
├── init          # Guided setup
├── index         # Index documents
├── search        # Search documents
├── status        # System status
├── doctor        # Health checks
├── config        # Configuration
├── reindex       # Re-index documents (v0.2.6)
├── list          # List documents (v0.2.7)
├── meta/         # Metadata subcommands (v0.2.7)
│   ├── show
│   └── edit
├── tag/          # Tag subcommands (v0.2.7)
│   ├── add
│   ├── remove
│   └── list
├── export        # Export archive (v0.2.8)
├── import        # Import archive (v0.2.8)
└── watch/        # Watch subcommands (v0.2.9)
    ├── start
    ├── stop
    └── status
```
