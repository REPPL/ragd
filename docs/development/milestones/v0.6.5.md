## Release Notes

### ragd v0.6.5 - Polish & Stability

**The details matter.**

This release focuses on user experience improvements and proactive error prevention.

**New Features:**

- **Configuration Validation** - `ragd config --validate` catches issues before they cause runtime errors
- **RAGAS LLM Metrics** - Faithfulness and answer relevancy scores with `--include-llm`
- **Visual Polish** - Progress bars, thinking spinners, ASCII banners
- **Better Errors** - User-friendly messages with install instructions

**Getting Started:**

```bash
# Validate your config
ragd config --validate

# Evaluate with LLM metrics
ragd evaluate --query "What is RAG?" --expected "..." --include-llm

# Watch the visual improvements
ragd init
ragd doctor
```

**What's Next:**

v0.7 focuses on privacy: PII detection and encryption for sensitive documents.
