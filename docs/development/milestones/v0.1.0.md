# Milestone: v0.1.0 - Core RAG

The foundation release establishing basic RAG functionality.

## Goal

Deliver a working RAG system that can index documents and perform semantic search via CLI. This milestone proves the core value proposition: users can add documents and find information using natural language queries.

## CLI Commands

| Command | Description |
|---------|-------------|
| `ragd init` | Guided first-run setup |
| `ragd index <path>` | Index documents |
| `ragd search "<query>"` | Search knowledge base |
| `ragd status` | View system status |
| `ragd doctor` | Run comprehensive health checks |
| `ragd --version` | Show version |
| `ragd --help` | Show help |

## Supported Formats

- PDF (clean, text-based)
- TXT (plain text)
- Markdown (.md)
- HTML (.html, .htm)

## Technology Stack

| Component | Technology |
|-----------|------------|
| CLI Framework | Typer + Rich |
| Vector Store | ChromaDB |
| Embeddings | sentence-transformers (all-MiniLM-L6-v2) |
| Text Extraction | PyMuPDF, BeautifulSoup |
| Configuration | Pydantic v2 |
| Testing | pytest |

## Success Criteria

- [x] User can index a PDF, TXT, MD, or HTML file
- [x] User can search indexed documents with natural language
- [x] Search returns relevant results with source attribution
- [x] Status command shows document count and storage usage
- [x] Health check validates all components
- [x] All operations work offline (no external API calls)
- [x] Test coverage > 80% for core functionality
- [x] Documentation includes getting started tutorial

## Non-Goals for v0.1

These are explicitly out of scope:

- Messy PDF handling (OCR, Docling) → v0.2
- Metadata management → v0.2
- Export/backup → v0.2
- Advanced search (hybrid, reranking) → v0.3
- Multi-modal (images) → v0.4
- Chat/LLM integration → v0.5
- Web UI → v1.0

## Draft Release Notes

### ragd v0.1.0 - Core RAG

**The foundation release.**

ragd is a privacy-first personal knowledge system. Index your documents and search them using natural language - all running locally on your machine.

**Features:**

- Index PDF, TXT, and Markdown files
- Semantic search with natural language queries
- Beautiful CLI with Rich output
- Guided setup with hardware detection (`ragd init`)
- Comprehensive health diagnostics (`ragd doctor`)

**Getting Started:**

```bash
pip install ragd
ragd init                              # Hardware detection + configuration
ragd index ~/Documents/notes/          # Index your documents
ragd search "how does authentication work"
```

**What's Next:**

v0.2 will add the killer feature: processing messy PDFs including scanned documents, complex layouts, and tables.
