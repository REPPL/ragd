## Focus Areas

### Fine-Tuning Pipeline (F-058)

Train custom models on your documents:

```bash
# Create fine-tuning dataset from indexed documents
ragd finetune prepare --output training_data.jsonl

# Fine-tune embedding model
ragd finetune train --base nomic-embed-text --epochs 3

# Use fine-tuned model
ragd config set embedding.model ./models/my-finetuned.gguf
ragd reindex --all
```

### Supported Frameworks

- **MLX** (Apple Silicon) - Native performance on M1/M2/M3
- **CUDA** (NVIDIA) - GPU acceleration for Linux/Windows
- **CPU** - Fallback for any platform

### Fine-Tuning Options

```bash
# Embedding model fine-tuning
ragd finetune train \
  --base nomic-embed-text \
  --dataset training_data.jsonl \
  --epochs 3 \
  --batch-size 32

# Validate fine-tuned model
ragd finetune evaluate --model ./models/my-finetuned.gguf

# Compare with base model
ragd finetune compare --base nomic-embed-text --finetuned ./models/my-finetuned.gguf
```

### Dataset Preparation

```bash
# Generate training pairs from indexed documents
ragd finetune prepare \
  --source indexed \
  --strategy semantic-pairs \
  --output training_data.jsonl

# Strategies available:
# - semantic-pairs: Similar document chunks
# - question-answer: Q&A from document content
# - triplets: Anchor-positive-negative
```
