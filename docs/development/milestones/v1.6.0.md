## Focus Areas

### Fine-Tuning Pipeline (F-058)

Train custom models on your documents:

```bash
# Create fine-tuning dataset from indexed documents
ragd finetune prepare --output training_data.jsonl

# Fine-tune embedding model
ragd finetune train --base nomic-embed-text --epochs 3

# Use fine-tuned model
ragd config set embedding.model ./models/my-finetuned.gguf
ragd reindex --all
```

### Workflow Automation (F-201)

Define repeatable workflows:

```bash
# Create a workflow
ragd workflow create "daily-digest" \
  --step "search new:today" \
  --step "summarise --length short" \
  --step "export --format markdown"

# Run workflow
ragd workflow run daily-digest

# Schedule workflow
ragd workflow schedule daily-digest --cron "0 8 * * *"
```

### Interactive Script Wizard (F-206)

Guided workflow creation for non-experts:

```bash
ragd workflow wizard
# ? What do you want to automate? [Search and summarise]
# ? How often should this run? [Daily at 8am]
# ? Where should output go? [~/reports/]
#
# Workflow 'daily-report' created!
```

### Shell Script Export (F-207)

Export workflows for use outside ragd:

```bash
ragd workflow export daily-digest --format bash > daily-digest.sh
ragd workflow export daily-digest --format python > daily_digest.py
```
