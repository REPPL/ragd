## Draft Release Notes

### ragd v0.7.0 - Privacy

**Trust your tools.**

Your documents are personal. ragd now has enterprise-grade privacy features.

**New Features:**

- **PII Detection** - Identify personal information before indexing
- **Encrypted Storage** - AES-256 encryption at rest
- **Session Management** - Separate contexts for different searches
- **Secure Deletion** - Properly remove sensitive documents
- **Data Tiers** - Control what gets indexed

**Getting Started:**

```bash
# Enable encryption (prompted for password)
ragd config set security.encryption enabled

# Scan for PII before indexing
ragd index ~/Documents/contracts/ --scan-pii

# Create isolated session
ragd session create work
ragd search "salary information" --session work

# Securely delete document
ragd remove document-id --secure
```

**What's Next:**

v0.8 brings intelligence: personas, temporal queries, and knowledge graphs.
