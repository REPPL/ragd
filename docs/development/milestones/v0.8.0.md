## Draft Release Notes

### ragd v0.8.0 - Intelligence

**Know you, know more.**

ragd learns from your knowledge, not just stores it.

**New Features:**

- **Knowledge Graph** - Understand relationships between concepts
- **Personas** - Adapt responses to your context
- **Temporal Queries** - "What did I read about auth last week?"
- **Concept Discovery** - Explore related topics
- **Smart Suggestions** - "You might also want to know..."

**Getting Started:**

```bash
# Build knowledge graph
ragd graph build

# Explore concept relationships
ragd graph explore "authentication"

# Set persona for responses
ragd config set chat.persona researcher

# Temporal queries
ragd search "notes from last month about security"
ragd search "what did I learn about APIs this year"
```
