## Focus Areas

### Advanced HTML Engine (F-098)
- JavaScript rendering for dynamic content
- SPA support (React, Vue, etc.)
- Configurable rendering timeout

### PDF Layout Intelligence (F-099)
- Multi-column layout detection
- Form field extraction
- Annotation preservation
- Complex table handling

### Indexing Self-Evaluation (F-105)
Critical innovation - the indexing pipeline tests itself:

```bash
# Run self-evaluation
ragd index --evaluate ~/Documents/

# Generate evaluation report
ragd evaluate compare ~/Documents/ --report

# Analyse logs for patterns
ragd evaluate analyze-logs --since "2024-01-01"
```

Features:
- Source-to-index comparison
- Log pattern detection
- Edge case identification
- Feedback learning loop

### Indexing Resilience
- Checkpoint progress during large batches
- Resume from last checkpoint on crash
- Content hash comparison for change detection
- Fuzzy duplicate detection

## Success Criteria
