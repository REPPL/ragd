## Focus Areas

### Embedding Privacy (F-059)
- Embedding vectors that can't be reverse-engineered
- Eguard transformation for privacy-preserving embeddings
- Differential privacy noise addition
- Configurable privacy levels

### GDPR Compliance (F-060)
- Complete data erasure across all stores
- Vector store deletion
- Metadata removal
- Graph entity removal
- Audit trail of deletions

### Privacy Dashboard (F-300)
```bash
ragd privacy show
# Privacy Settings:
#   Embedding privacy: HIGH (Eguard + noise)
#   PII detection: ENABLED
#   Anonymisation: AUTO
#   Data lineage: ENABLED
#
# Data Summary:
#   Documents: 1,234
#   With PII detected: 45
#   Anonymised: 45
```

### Data Lineage (F-301)
Track where each piece of data came from:
- Source file path
- Import timestamp
- Processing steps applied
- Anonymisation history
