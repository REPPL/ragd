# Milestone: v1.0.0 - Performance & Polish

Production-ready release with performance optimisation and final polish.

## Goal

Make ragd production-ready with comprehensive performance profiling, memory optimisation, faster startup times, and published benchmarks. This release marks the transition from alpha to stable.

## Theme

**"Production Ready"** - Fast, stable, reliable.

## Draft Release Notes

### ragd v1.0.0 - Performance & Polish

**Production ready.**

After extensive alpha testing, v1.0 delivers stability and performance.

**What's New:**

- **Performance Profiling** - Understand and optimise your workflows
- **Memory Optimisation** - Lower resource usage for large collections
- **Faster Startup** - CLI responds instantly
- **Model Comparison** - Evaluate different models side-by-side
- **Backend Migration** - Seamlessly switch vector stores

**Complete Feature Set:**

ragd 1.0 includes everything from the journey:
- Core RAG (v0.1)
- Messy PDF handling (v0.2)
- Advanced search (v0.3)
- Multi-modal (v0.4)
- Chat with LLM (v0.5)
- Flexible storage (v0.6)
- Privacy features (v0.7)
- Knowledge graphs (v0.8)
- Enhanced indexing (v0.9)

**Getting Started:**

```bash
# Install (all runtime features included by default)
pip install ragd

# Expert minimal install (CI/resource-constrained)
RAGD_MINIMAL=1 pip install ragd

# Use CLI
ragd index ~/Documents/
ragd search "how does it work"
ragd chat
```

**What's Next:**

- v1.1: Knowledge Graph CLI
- v1.2: Onboarding
- v1.3: REST API
- v1.4: Workflow Automation
- v1.5: Web UI Foundation
- v1.6: Fine-Tuning
