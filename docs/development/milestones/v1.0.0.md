## Draft Release Notes

### ragd v1.0.0 - Personal Platform

**Your knowledge, your way.**

After nine releases building features, v1.0 brings it all together.

**What's New:**

- **Web Interface** - Access ragd from any browser
- **REST API** - Build your own integrations
- **Onboarding** - Guided first-run experience
- **Complete Documentation** - Everything you need to know

**Complete Feature Set:**

ragd 1.0 includes everything from the journey:
- Core RAG (v0.1)
- Messy PDF handling (v0.2)
- Advanced search (v0.3)
- Multi-modal (v0.4)
- Chat with LLM (v0.5)
- Flexible storage (v0.6)
- Privacy features (v0.7)
- Knowledge graphs (v0.8)

**Getting Started:**

```bash
# Install (all runtime features included by default)
pip install ragd

# Expert minimal install (CI/resource-constrained)
RAGD_MINIMAL=1 pip install ragd

# Start web UI
ragd serve
# Open http://localhost:8080

# Or use CLI as always
ragd index ~/Documents/
ragd search "how does it work"
ragd chat
```

**What's Next:**

ragd is now feature-complete for personal use. Future development will focus on community features, plugins, and specialised domains.
