# Milestone: v0.4.0 - Multi-Modal

Visual understanding for documents with images, diagrams, and charts.

## Goal

Extend ragd to understand visual content in documents. Using ColPali/ColQwen vision embeddings, users can search images, diagrams, charts, and screenshots as naturally as text.

## Theme

**"See and Search"** - Knowledge isn't just text.

## Use Cases Addressed

| Use Case | Description | Status |
|----------|-------------|--------|
| Image Search | Find images by description | ✅ Complete |
| Diagram Understanding | Search flowcharts, architecture diagrams | ✅ Complete |
| Chart Data | Query chart content semantically | ✅ Complete |
| Screenshot Search | Find UI screenshots and mockups | ✅ Complete |

## Features Included

| Feature | Description | Status |
|---------|-------------|--------|
| F-019: Multi-Modal Support | ColPali vision embeddings | ✅ Complete |
| Image Extraction | Extract images from PDFs | ✅ Complete |
| Visual Query | Search with image descriptions | ✅ Complete |
| Caption Generation | Auto-generate image descriptions | ✅ Complete |

## Dependencies

- **v0.3.0** - Advanced Search must be complete

## Technology Stack

| Component | Technology |
|-----------|------------|
| Vision Embeddings | ColPali / ColQwen2 |
| Image Extraction | PyMuPDF, Docling |
| Caption Model | LLaVA via Ollama (optional) |
| Vector Storage | ChromaDB (multi-modal collection) |

## Success Criteria

- [x] Images extracted and indexed automatically
- [x] Visual queries return relevant images
- [x] Diagrams searchable by content/description
- [x] Performance acceptable (< 1s per image embedding)
- [x] Works offline with local models

## Non-Goals for v0.4

- Video processing → Future
- Audio/transcription → Future
- Chat/LLM integration → v0.5
- Web UI → v1.0

## Draft Release Notes

### ragd v0.4.0 - Multi-Modal

**See and search.**

Your knowledge includes diagrams, charts, and screenshots. Now ragd understands them too.

**New Features:**

- **Vision Embeddings** - ColPali for visual understanding
- **Image Search** - Find images by description
- **Diagram Support** - Search flowcharts and architecture diagrams
- **Auto-Captioning** - Generate descriptions for images

**Getting Started:**

```bash
# Index documents with images
ragd index ~/Documents/presentations/

# Search for visual content
ragd search "system architecture diagram"
ragd search "screenshot of login page"
ragd search "chart showing revenue growth"
```

**What's Next:**

v0.5 brings chat: have a conversation with your knowledge base.
