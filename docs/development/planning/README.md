# Planning Directory

This directory contains **high-level design documents** that inform feature development.

## What Belongs Here

- **Interface contracts** - Protocol definitions and API boundaries
- **Architecture visions** - High-level system design
- **Requirements documents** - User needs and constraints
- **Design goals** - What we're trying to achieve and why

## What Does NOT Belong Here

| Document Type | Correct Location |
|--------------|------------------|
| Feature specifications | `features/` |
| Release planning | `milestones/` |
| Implementation records | `implementation/` |
| Research findings | `research/` |

## Key Distinction

- **Planning documents** describe **what** to build and **why**
- **Feature specifications** describe **how** and **when** to build
