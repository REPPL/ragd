# Images

Image assets for documentation.

## Contents

| File | Purpose |
|------|---------|
| `ragd-logo.png` | Project logo |

## Guidelines

- **Logos**: PNG format, transparent background where appropriate
- **Screenshots**: PNG format, reasonable dimensions
- **Diagrams**: SVG preferred for scalability, PNG acceptable
- **Maximum size**: Keep under 500KB per image

## Naming Convention

Use kebab-case with descriptive names:
- `feature-name-diagram.png`
- `screenshot-cli-help.png`
- `architecture-overview.svg`
