# AI Image Generation Prompts

This directory contains prompts for generating images using AI tools like Google Gemini, DALL-E, or Midjourney.

## Purpose

- Store reproducible prompts for project visuals
- Enable anyone to regenerate images consistently
- Document the creative direction for marketing materials

## Contents

| File | Description |
|------|-------------|
| `getting-started-comic.md` | Comic tutorial showing new users how to get started with ragd |

## Usage

1. Copy the prompt from the relevant file
2. Paste into your AI image generation tool
3. Adjust parameters as needed (aspect ratio, style, etc.)
4. Store generated images in `docs/assets/img/` with descriptive names

## Contributing

When adding new prompts:
- Use descriptive filenames
- Include context about the target audience and style
- Note which AI tool the prompt was designed for
