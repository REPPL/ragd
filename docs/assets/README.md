# Documentation Assets

Static assets for documentation including images, diagrams, and media files.

## Structure

| Directory | Purpose |
|-----------|---------|
| [img/](./img/) | Images, logos, and screenshots |

## Guidelines

- Use descriptive kebab-case filenames (e.g., `feature-diagram.png`)
- Optimise images for web (< 500KB per image where possible)
- Supported formats: PNG, JPG, SVG
- Reference using relative paths from markdown files

## Adding Assets

1. Place files in the appropriate subdirectory
2. Use meaningful names that describe the content
3. Update documentation to reference the new asset
