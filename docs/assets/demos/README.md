## Recording Standards

### Terminal Setup
- **Size:** 100x30 characters
- **Font:** Monospace, 14pt
- **Theme:** Dark background, light text
- **Shell:** Clean prompt, no personal info

### Recording Tools
- [asciinema](https://asciinema.org/) - Terminal recording
- [vhs](https://github.com/charmbracelet/vhs) - Automated terminal GIFs
- [terminalizer](https://terminalizer.com/) - GIF generation

### Quality Guidelines
- Clear, readable text
- Pause on important commands
- Highlight key outputs
- No personal information visible
