# Chat Tour Demo

A 2-3 minute video walkthrough of ragd's chat interface.

## Demo Script

### Scene 1: Introduction (15s)

"Let's explore ragd's conversational chat interface..."

### Scene 2: Starting Chat (20s)

```bash
ragd chat
```

Show the chat interface startup.

### Scene 3: Basic Questions (45s)

```
> What authentication methods are documented?
```

Show the response with citations.

```
> How do I configure OAuth2?
```

Show follow-up response maintaining context.

### Scene 4: Multi-Turn Conversation (45s)

```
> What are the security considerations?

> Can you summarise the key points?
```

Demonstrate context retention across turns.

### Scene 5: Chat Commands (30s)

```
> /sources
> /clear
> /export chat-log.md
```

Show in-chat commands.

### Scene 6: Wrap-up (15s)

"ragd chat brings conversational AI to your personal documents."

## Recording Notes

- **Duration:** 2-3 minutes
- **Terminal size:** 100x30
- **Pause:** 3s after responses
- **Highlight:** Citation display
